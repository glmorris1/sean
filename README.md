# Sean

Sean is a static desktop browser-style trading terminal inspired by charting workspaces. It includes a browser chrome, watchlist, drawing toolbar, candlestick chart, market details, order ticket, and bottom trading dock.

## Run Locally

Open `index.html` in a browser, or serve the folder with any static server:

```sh
python3 -m http.server 4173
```

## GitHub Pages

This project is designed to publish from the repository root on the `main` branch.
